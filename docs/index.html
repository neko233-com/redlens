Full-Stack Security Assessment

Scan your infrastructure for vulnerabilities, generate compliance-ready reports with full evidence chains.

Get Started View on GitHub

Features

Web Vulnerability Scanning

Detect SQL injection, XSS, admin panel exposure, directory listing, and missing security headers.

Network Security Audit

Port scanning, service fingerprinting, weak credential detection for databases and services.

Host Configuration Check

OS security settings, exposed services, DNS misconfigurations, and privilege analysis.

Full Evidence Chain

Reports include credentials, screenshots, PoC scripts, timestamps, and impact analysis.

Compliance Reports

Generate HTML and JSON reports mapped to OWASP Top 10 and CIS benchmarks.

Docker Simulation

Test against intentionally vulnerable applications with Docker Compose orchestration.

Installation

# Build from source
git clone https://github.com/redlens/redlens.git
cd redlens
go build -o redlens ./cmd/redlens

# Or use Docker
docker build -t redlens .
docker run -p 8080:8080 redlens

Usage

# Start the server
./redlens serve

# Open browser
# http://localhost:3000

Docker Simulation

# Run with vulnerable test app
cd docker
docker-compose up --build

# redlens will scan the vulnerable web app
